# 更新日志

## Unreleased

- Paste剪切板首次运行后自动登记为随 ZTools 启动，并保持后台剪贴板监听。

## 1.0.0 - 2026-07-20

- 首次发布 Paste剪切板双宿主插件。
- 支持剪贴板历史、搜索、分组、预览、快捷粘贴与连续粘贴队列。
- 支持图片缩略图、PDF、Quick Look、图片旋转和 macOS Vision OCR。
- 支持上、下、左、右贴边、多屏跟随、紧凑布局和失焦关闭。
- 支持历史保留、附件预算单位、隐私规则、屏幕共享保护与暂停捕获。
- 支持端到端加密 WebDAV 同步和跨宿主冲突合并。
- 增加双宿主视觉矩阵、搜索性能门禁、helper 证明和最终 ZIP 校验。
- 品牌名称更新为“Paste剪切板”，启用全新紫蓝渐变图标。
