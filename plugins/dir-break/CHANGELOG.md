# Changelog

## [1.0.0] - 2026-07-15

###  新增
- 拖入文件夹或文本指令触发解散
- 单层/递归两种解散模式
- 三种文件名冲突策略：自动重命名、覆盖、跳过
- 文件预览树展示受影响文件
- 一键撤回解散操作
- 历史记录持久化存储（最近 50 条）
- 冷静蓝扁平圆角设计，支持暗色主题
- 目录也可以解散到上一层：子目录作为整体移动到父目录（单层模式），不再只提文件
- 递归模式下：文件提取到父目录后，非空子目录也会被整体移出
- 结果预览适配：目录列表标注"将移到父目录" / "将被清空"
- 操作摘要显示目录移动数量（如：移动 3 个目录）
- 结果预览全面升级：统计栏（文件数/目录数/总大小）、文件类型分布徽标
- 前后对比模块：直观展示解散前文件分布 vs 解散后结构变化
- 大量文件/目录的安全处理：限制预览数量，点击展开全部

###  修复
- 修复切换模式按钮颜色反转问题：选中为蓝色，未选中为白色（原因为 `handleToggleRecursive` 函数内部重复取反导致状态混乱）
