## 插件介绍
- 奶龙插件是一个基于ztools的插件，打开后桌面会出现一个奶龙图案。
- ![奶龙插件](https://origin.picgo.net/2026/07/22/1fomat38e0b9db84309bc7.png)

正常情况奶龙眼睛会动。这个只是效果。
鼠标单击后，奶龙会发出魔性笑声。
右键奶龙，插件直接退出。
