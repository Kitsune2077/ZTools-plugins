# 封印解除 Logo 设计

## 概述

为 ZTools 插件「封印解除」(unlock-file) 设计品牌 Logo。该插件用于检测占用文件/目录的进程并一键结束，同时包含文件粉碎和端口检测功能，仅支持 Windows。

---

## 1. 设计方案

### 选定方案：「锁开一瞬间」

一把钥匙插入锁孔转动的瞬间，锁梁弹开，锁体迸发出「解除封印」的解放感。

### 备选方案

| 方案 | 描述 |
|------|------|
| A — 钥匙君 | 拟人化小钥匙角色，圆脑袋大眼睛，俏皮表情 |
| C — 封印碎碎 | 圆形封印阵/锁裂纹碎开，碎片飞散，偏涂鸦风格 |

---

## 2. 视觉风格

- **风格**：轻松有趣 (fun/cute)，卡通风格 (cartoon-style)，避免写实或工业感
- **表情**：锁无表情，钥匙可以有得意的 wink 表情（😏），两点一弧即可
- **构图**：居中对称或略微倾斜（增加动感），锁占主体约 60%，钥匙从右侧或下方插入
- **动感**：锁梁朝上弹开，配合 2-3 条简单的速度线/迸发线条强化动态
- **比例**：在方形区域内构图，适配 ZTools 插件圆形/方形图标裁剪

---

## 3. 色彩

| 色名 | 色值 | 用途 |
|------|------|------|
| 主色 | `#42b883` | 锁/钥匙主体色，代表「已解放/安全」 |
| 辅助色 1 | `#58a4f6` | 迸发特效、速度线、光效 |
| 辅助色 2 | `#ffffff` | 高光、钥匙齿细节 |
| 辅助色 3 | `#2a2a2a` | 轮廓线/阴影，与插件深色 UI 融合 |

**配色逻辑**：锁和钥匙主体用 `#42b883` 绿色，锁孔/钥匙齿局部用深色切割出层次感。解锁迸发的特效光用蓝色，营造「封印解除」的魔法感。

---

## 4. 字体

推荐圆润无衬线字体（用于插件标题、品牌文字等场景）：

- **Nunito** — 圆润现代，英文场景
- **Noto Sans SC** — 中文字体，与 Nunito 风格统一
- **ZCOOL KuaiLe** — 更活泼的卡通感

## 5. 使用场景 & 衍生尺寸

| 场景 | 尺寸 | 说明 |
|------|------|------|
| 主图标（插件市场/plugin.json） | 256×256px | 完整构图：锁 + 钥匙 + 弹开光效 |
| 小尺寸标记 | 64×64px / 32×32px | 简化版：只保留钥匙插入锁孔特写，裁掉弹开的锁梁 |
| 功能 tab 图标 | 24×24px | 纯钥匙符号，作为三个 tab 的统一品牌标记 |
| File Explorer 右键图标 | 16×16px | 极简版本：钥匙头部环形 + 一点齿 |

**格式要求**：SVG（矢量源文件）+ PNG（@1x、@2x、@3x）。
