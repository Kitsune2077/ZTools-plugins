# 解除占用

ZTools 插件 — 解除文件占用、粉碎文件、检测端口占用。仅支持 Windows。

## 使用方式

### 关键字启动

在 ZTools 搜索框输入：

| 关键字 | 功能 |
|---|---|
| `解除占用` | 打开解除占用标签页 |
| `文件粉碎` | 打开文件粉碎标签页 |
| `端口检测` | 打开端口检测标签页 |

### 拖拽启动

将文件或目录拖到 ZTools 搜索框，选择「解除占用」即可直接带入路径并自动扫描。

## 功能

### 解除占用

检测哪些进程正在占用文件或目录，支持一键结束。


### 文件粉碎

安全删除文件或目录，粉碎模式会先用随机数据覆写再删除。


### 端口检测

查看指定端口的监听进程，支持一键结束。
